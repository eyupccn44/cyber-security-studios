# Reports Directory

Completed, client-ready security deliverables are stored here.

## Naming Convention

```
[CLIENT]-[YYYY-MM-DD]-[TYPE]-report.md

Examples:
acme-corp-2024-01-15-webapp-pentest-report.md
techco-2024-02-20-red-team-report.md
startup-2024-03-10-compliance-gap-iso27001.md
```

## Report Types

- `*-pentest-report.md` — Penetration test findings
- `*-red-team-report.md` — Red team operation narrative
- `*-ir-report.md` — Incident response post-mortem
- `*-compliance-gap.md` — Compliance gap analysis
- `*-risk-assessment.md` — Risk assessment report
- `*-exec-summary.md` — Standalone executive summary

## Access Control

All reports are CONFIDENTIAL.
Only share via encrypted, client-approved channels.
Do not include raw captured credentials in reports.
